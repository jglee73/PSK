
#. Wafer Transfer Sequence 관련
 
1) Metal -> Strip Chamber 진입 시 Robot이 Wafer를 들고있는 경우 진입 X 
  * Recipe가 다른 경우 Wafer는 같은 TM에 존재하면 안됨
  * 같은 Recipe가 Process 되는 경우 Chamber 내 같이 이동 가능
-> 완료 (SAME.PROCESS)

2) 두개의 Chamber의 Strip Process 대기 시간이 얼마 남지 않은 경우 Chamber에서 Wait후 Robot이 두장 다 LL으로 이동
-> 완료 (Config 항목 추가 : "Wafer waiting limit time on arm")
  

#. Wafer Aligner 각도 설정
  * 현재 Aligner Target각도가 LL기준으로 되어있으나, PM기준으로 변경이 필요합니다. 
  * PM은 Metal을 기준으로 Align이 필요합니다. (A,B,E,F Chamber)
-> 완료 (TMC Test)
-> 작업 중 (CTC Scheduler) 

#. EFEM Robot Arm A, B 위치 변경
  * A Lower로 변경 (현재 Upper Arm이 A로 되어있음)
  * B Upper로 변경 (현재 Lower Arm이 B로 되어있음)
-> 완료  

#. EFEM Robot Dual Pick/Place
  * Dual Arm 사용 하는 경우 Pick/Place 시 사용 Arm을 11로 사용하여 Command 전송
  * LP 24, 25 Slot을 집을 때 Arm 11로 Dual Arm 선택 후, 24 Slot으로 Command 전송해야 24, 25 Pick& Place 가능
  * Single Stage는 Single Arm만 사용해서 Pick/Place 진행
-> 완료 (TMC Test)
-> 작업 중 (CTC Scheduler) 
 
#. ATM Robot Servo Off 상태에서도 Command Complete (Ex, Home Roate)
   ATM Robot이 Servo Off 상태에서 동작 전송시, 실제 동작은 수행하지 않지만 Command Complete되는 문제 발생
  * Servo 상태 확인 
  * Off 상태이면 Alarm Post 
-> 완료
